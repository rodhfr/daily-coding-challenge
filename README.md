## Structure

Each file is named following the pattern:

```
<company>_<problem_description>.py
```

To include topics and difficulty in the table, add these comments anywhere in
the first 20 lines of the file:

```python
# topics: Arrays, Hash Set
# difficulty: Easy
```

## Running

```bash
python3 <filename>.py
```
